# YourTube

**YourTube** - приложение для скачивания видео с YouTube, RuTube и других платформ
>v1.0 предназначена для тестирования. Релиз будет начиная с 1.1!

## ✨ О программе

- Понятный и минималистычный интерфейс
- Вывод логов прямо внутри программы
- Встроеная обход защиты от ботов (Для YouTube)

## ⚙️ Установка

YourTube v1.0 является пререлизом. Чтобы использовать программу, необходимо скачать PyCharm, Python 3.12.2 и выше, скачать проект и запустить.
