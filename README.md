## ✏️ Изменено

- Добавлена иконка программы
- Возможность управления логами (теперь только пользователь решает сохранять их в файл или нет),
- Исправлена информация об окнах
- Добавлен выбор тем
> Темы выбираются при скачивании программы. Побробнее во вкладке Releases!

## ✨ О программе

- Понятный и минималистычный интерфейс
- Вывод логов прямо внутри программы
- Возможность сохранения логов
- Скачивание аудио (только .mp3)
- Встроенный обход защиты от ботов (Для YouTube)

## ⚙️ Установка

Для использования YourTube достаточно скачать один из установочников (разные установочники - разное оформление, см. в "Releases"), запустить и пользоваться=)
