# YourTube

**YourTube** - приложение для скачивания видео с YouTube, RuTube и других платформ

## ✏️ Изменено

- Добавлена иконка программы
- Возможность управления логами (теперь только пользователь решает сохранять их в файл или нет),
- Исправлена информация об окнах
- Добавлен выбор тем
> Темы выбираются при скачивании программы. Побробнее во вкладке Reliase!

## ✨ О программе

- Понятный и минималистычный интерфейс
- Вывод логов прямо внутри программы
- Встроенный обход защиты от ботов (Для YouTube)

## ⚙️ Установка

YourTube v1.0 является пререлизом. Чтобы использовать программу, необходимо скачать PyCharm, Python 3.12.2 и выше, скачать проект и запустить.
